# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in the Aegleseeker Client, please follow these steps:

1. **Do not disclose publicly.**
2. Contact the project maintainer via Discord: `notvyzer`.
3. Provide details including:

   * Steps to reproduce the issue
   * Affected version(s)
   * Any relevant screenshots or logs

We will acknowledge your report within 48 hours and work on a fix as soon as possible.

---

## Supported Versions

* Minecraft: Windows 10 Edition Beta 0.15.10
* Aegleseeker Client v1.1.61

---

## Security Updates

* All security fixes will be included in new releases.
* Users are encouraged to always use the latest version.

---

## Reporting Other Issues

For non-security issues, please use GitHub Issues or Discord to submit feedback, feature requests, or bug reports.
