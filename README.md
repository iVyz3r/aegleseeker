## Installation

### Step 1

Install Git Bash: [click here](https://git-scm.com/downloads)

### Step 2

Open CMD or PowerShell and execute this command:

```bash
git clone https://github.com/iVyz3r/aegleseeker.git
```

### Step 3

Open the project:

```bash
cd aegleseeker
```

### Step 4

Execute **build.bat** or **ps-build.ps1**
